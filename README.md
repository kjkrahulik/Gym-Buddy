# Gym-Buddy

## Overview
Gym-Buddy is a comprehensive fitness and workout management application designed to help users track their workouts, create custom workout templates, set fitness goals, and connect with friends. The application features a community-focused social platform where users can share their fitness journey, compete on leaderboards, and support each other in achieving their health goals.

## Features
- **Workout Management**: Create and customize workout templates with exercises from a comprehensive exercise database
- **Progress Tracking**: Log workout sessions and monitor your fitness progress over time
- **Goal Setting**: Set personalized fitness goals and track progress toward achieving them
- **Social Features**: Connect with friends, send invitations, and view friend profiles
- **Leaderboards**: Compete with friends on global fitness leaderboards
- **Account Management**: Secure user authentication and personalized account settings
- **Exercise Database**: Access a curated library of exercises with muscle group categorization

## Technology Stack
- **Backend**: Java Spring Boot
- **Database**: PostgreSQL
- **Frontend**: HTML, CSS, JavaScript with Thymeleaf templating
- **Build Tool**: Maven
- **Architecture**: MVC (Model-View-Controller) with RESTful APIs

## Getting Started
1. Clone the repository
2. Navigate to the `app` directory
3. Ensure PostgreSQL is installed and running
4. Run `mvn clean install` to build the project
5. Run the application using Maven: `mvn spring-boot:run`
6. Access the application at `http://localhost:8080`

For database setup instructions, see [POSTGRESREADME.md](app/POSTGRESREADME.md)
